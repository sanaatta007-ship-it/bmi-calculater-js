# BMI Calculator

## 📌 Overview

This project is a JavaScript-based BMI Calculator that calculates Body Mass Index (BMI) using a user's height and weight.

It also classifies the calculated BMI into one of the standard health categories.

## ✨ Features

- Calculate Body Mass Index (BMI)
- Display BMI category
- Underweight classification
- Normal weight classification
- Overweight classification
- Obese classification

## 🛠️ Technologies Used

- JavaScript
- HTML
